### COPILOT settings

```json
  "chat.tools.terminal.terminalProfile.osx": {
    "path": "zsh",
    "env": { "IS_AGENT": "1" }
  },
  "chat.tools.terminal.terminalProfile.linux": {
    "path": "zsh",
    "env": { "IS_AGENT": "1" }
  }
```

This setting is used to not configure starship and atuin when it is an agent terminal.
